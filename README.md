#  SimpleYtdl

**Download YouTube and other social media videos easily, without any complexity.**

---

##  Run the App

You can run SimpleYtdl as a standard desktop application or test it directly in your web browser.

**Desktop (Windows, Linux, or macOS):**
```bash
flet run

```

**Web App:**

```bash
flet run --web

```

> **Tip:** For more details on running Flet applications, refer to the [Getting Started Guide](https://docs.flet.dev/).
